# SEO最適化プロンプト

POD・転売商品のタイトル・タグ・説明文をSEO最適化するテンプレート集。

## キーワードリサーチ

```xml
<role>ECプラットフォームのSEOスペシャリストです。
Shopify、Amazon、Etsy、メルカリのアルゴリズムを熟知しています。</role>

<task>
以下の商品カテゴリーで、購買意欲の高い顧客が検索するキーワードを抽出してください。
カテゴリー: [カテゴリー名]
プラットフォーム: [Shopify / Amazon / Etsy / メルカリ]
ターゲット市場: [英語圏 / 日本]
</task>

<output_format>
1. 高ボリュームキーワード（月間1万回以上）: [5個]
2. 中ボリュームキーワード（1,000〜10,000回）: [5個]
3. 長尾キーワード（低競合・高CVR）: [10個]
4. 競合が使っていない隙間キーワード: [3個]
</output_format>
```

## タグ最適化（Etsy/Shopify向け）

```xml
<task>
以下の商品に最適なタグを20個生成してください（Etsy上限20タグ）。
商品: [商品名・デザイン説明]
ニッチ: [ターゲットニッチ]
</task>

<constraints>
- 各タグ20文字以内
- 一般的すぎるタグ（例: "tshirt"）は3個まで
- 具体的なニッチタグを多めに
- 季節・イベント関連タグも含める
- 商標を含まない
</constraints>
```

## 商品タイトルA/Bテスト生成

```xml
<task>
同じ商品で検索意図の異なる3パターンのタイトルを生成してください。
商品: [商品名]

パターンA: 機能訴求型（何ができるか）
パターンB: 感情訴求型（どんな気持ちになるか）
パターンC: ニッチキーワード型（特定の趣味・コミュニティ向け）
</task>
```
