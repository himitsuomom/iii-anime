# 商品説明文生成プロンプト

LLM Wiki に準拠した Output Shape 指定形式。

## 基本テンプレート

```xml
<role>あなたはShopifyおよびPOD商品販売に精通したECコピーライターです。
SEO最適化・高コンバージョン・ブランドトーンを両立した商品説明文を書きます。</role>

<context>
プラットフォーム: Shopify（英語圏向け） / メルカリ（日本向け）
商品カテゴリー: [カテゴリー名]
ターゲット: [年齢層・性別・趣味等]
競合価格帯: [価格帯]
</context>

<task>
以下の商品情報から、SEO最適化された商品説明文を生成してください。
商品名: [商品名]
素材/仕様: [スペック]
デザインコンセプト: [説明]
</task>

<constraints>
- タイトル: 80文字以内（Shopify推奨）
- 本文: 150〜300文字
- 箇条書き: 3〜5点（主要メリット）
- SEOキーワード: 自然に3〜5個を含める
- 誇大表現・著作権侵害リスクのある表現を使わない
- 絵文字は使わない（日本向け以外）
</constraints>

<output_format>
以下の形式で出力してください：
---
タイトル: [タイトル]
説明文: [本文]
箇条書き:
- [メリット1]
- [メリット2]
- [メリット3]
SEOタグ: [カンマ区切りで5〜10個]
---
</output_format>

<thinking>タグの中でSEOキーワードと訴求ポイントを検討してから、説明文を書いてください。
```

## 日本向け（メルカリ・Yahoo!オークション）バリアント

```xml
<role>メルカリ・Yahoo!オークションで高評価を獲得している出品者です。
購入者の検索行動を理解し、信頼感のある商品説明を書きます。</role>

<task>
以下の商品を日本向けに出品する説明文を生成してください。
商品: [商品名]
状態: [新品/中古/詳細]
付属品: [付属品リスト]
</task>

<constraints>
- タイトル: 40文字以内
- 説明文: 200〜500文字
- 状態の正確な記述を含める
- 発送方法・梱包方法を明記
- 「即購入OK」等の購買促進フレーズを含める
</constraints>
```

## Self-Refine（品質向上）用レビュープロンプト

生成後に以下でレビューをかける：

```
生成した商品説明文を以下の観点でレビューしてください：
- SEOキーワードが自然に含まれているか
- 著作権・商標侵害リスクのある表現はないか
- ターゲット層に刺さる訴求になっているか
- 誤字・不自然な日本語/英語はないか

問題点をリストアップし、改善版を生成してください。
```
